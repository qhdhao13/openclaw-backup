## 港股交易日历

接口：hk_tradecal描述：获取交易日历限量：单次最大2000权限：用户积累2000积分才可调取

输入参数

<table>
<thead>
<tr>
<th>名称</th>
<th>类型</th>
<th>必选</th>
<th>描述</th>
</tr>
</thead>
<tbody><tr>
<td>start_date</td>
<td>str</td>
<td>N</td>
<td>开始日期</td>
</tr>
<tr>
<td>end_date</td>
<td>str</td>
<td>N</td>
<td>结束日期</td>
</tr>
<tr>
<td>is_open</td>
<td>str</td>
<td>N</td>
<td>是否交易 '0'休市 '1'交易</td>
</tr>
</tbody></table>
输出参数

<table>
<thead>
<tr>
<th>名称</th>
<th>类型</th>
<th>默认显示</th>
<th>描述</th>
</tr>
</thead>
<tbody><tr>
<td>cal_date</td>
<td>str</td>
<td>Y</td>
<td>日历日期</td>
</tr>
<tr>
<td>is_open</td>
<td>int</td>
<td>Y</td>
<td>是否交易 '0'休市 '1'交易</td>
</tr>
<tr>
<td>pretrade_date</td>
<td>str</td>
<td>Y</td>
<td>上一个交易日</td>
</tr>
</tbody></table>
接口示例

```

pro = ts.pro_api()

df = pro.hk_tradecal(start_date='20200101', end_date='20200708')


```
